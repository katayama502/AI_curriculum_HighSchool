Git / GitHub チートシート

📦 基本操作

$ git init              # 初期化
$ git status           # 状態確認
$ git add .            # 全ファイルをステージ
$ git commit -m "msg"  # 変更を記録
$ git log --oneline    # 履歴を表示
$ git diff             # 差分を表示

🌿 ブランチ操作

$ git branch           # ブランチ一覧
$ git checkout -b name # 作成＋移動
$ git checkout main    # mainに切り替え
$ git merge feature    # ブランチをマージ
$ git branch -d name  # ブランチを削除

☁️ リモート（GitHub）

$ git remote add origin [URL]
$ git push origin main  # アップロード
$ git pull origin main  # ダウンロード
$ git clone [URL]       # コピーして取得

🔙 取り消し・修正

# 最後のコミットメッセージを修正
$ git commit --amend

# ステージを取り消す
$ git restore --staged file

# 過去のコミットに戻る
$ git revert コミットID

Git はエラーが怖くない！コマンドの意味がわからないときは「git [コマンド] の使い方を教えて」とClaudeに聞くと一発解決。
