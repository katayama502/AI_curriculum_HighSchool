Props — コンポーネントへのデータ渡し

Propsとは？

Propsは「親コンポーネントから子コンポーネントへデータを渡す仕組み」。関数の引数のようなもの。
コンポーネントとPropsの使い方
// 子コンポーネント: Propsを受け取る
function WeatherCard({ city, temp, icon }) {
return (
<div className="card">
<span>{icon}</span>
<h2>{city}</h2>
<p>{temp}°C</p>
</div>
);
}

// 親コンポーネント: Propsを渡す
function App() {
return (
<div>
<WeatherCard city="東京" temp={25} icon="☀️" />
<WeatherCard city="大阪" temp={28} icon="🌤️" />
</div>
);
}

親 → 子へ一方向に流れる

Propsの型
文字列: city="東京"
数値: temp={25}
真偽値: isSunny={true}
配列: cities={["東京","大阪"]}
