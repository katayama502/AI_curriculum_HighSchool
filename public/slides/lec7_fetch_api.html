外部データを取得する（fetch API）

インターネットからデータを受け取る

1

fetchでリクエスト

URLを指定してデータを要求する

2

Promiseが返ってくる

非同期なので「後で結果が来る」という約束

3

.json()でデータを取り出す

受け取ったデータをJavaScriptで使える形に変換

// async/awaitを使った書き方（現代的） async function getWeather() { const res = await fetch( "https://api.example.com/weather" ); const data = await res.json(); // データをHTMLに表示 document.getElementById("city") .textContent = data.city; document.getElementById("temp") .textContent = data.temperature + "℃"; }

APIとは？

外部サービスからデータをもらうための「窓口」。天気、地図、翻訳など様々なAPIが無料で使えます。

JSONとは？

データのやり取りに使われる形式。
{"name":"太郎","age":16}
JSのオブジェクトに似た見た目です。

async/awaitとは？

時間がかかる処理（通信など）を待つための書き方。awaitを付けると、結果が来るまで次の行を実行しません。
