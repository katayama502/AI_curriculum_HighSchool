HTMLの基本構造を理解する

すべてのHTMLファイルの基本形

<!DOCTYPE html> ← HTML5宣言 <html lang="ja"> ← ページ全体 <head> ← ブラウザへの情報 <meta charset="UTF-8"> <title>ページタイトル</title> </head> <body> ← 表示されるコンテンツ <h1>見出し</h1> <p>段落テキスト</p> </body> </html>

重要タグの役割

  • <html>ページ全体を囲む最外殻タグ
  • <head>タイトル・CSSなど、ブラウザ向け設定
  • <body>実際に画面に表示される内容
  • <h1>最大見出し（h1〜h6まである）
  • <p>段落（パラグラフ）テキスト
ルール：タグは必ず「閉じる」
<タグ名> で開いたら </タグ名> で閉じる。これを忘れるとレイアウトが崩れる。
