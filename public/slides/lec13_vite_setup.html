Vite + React のセットアップ

コマンド一発で始められる

$ npm create vite@latest my-app -- --template react
✔ Project created!

$ cd my-app
$ npm install
added 200 packages...

$ npm run dev
  ➜  Local:   http://localhost:5173/
my-app/
├── src/
│   ├── App.jsx      ← ここを編集
│   ├── main.jsx     ← エントリーポイント
│   └── App.css
├── index.html
└── package.json

各ステップの説明

1
npm create vite@latest
Viteのテンプレートからプロジェクトを自動生成。react を選択するだけ。
2
npm install
ReactなどのライブラリをダウンロードしてNode.jsに読み込ませる。
3
npm run dev
開発サーバーが起動。ブラウザで localhost:5173 を開いて確認できる。
★
src/App.jsx を自由に編集
保存するたびブラウザが自動更新（Hot Module Replacement）。
