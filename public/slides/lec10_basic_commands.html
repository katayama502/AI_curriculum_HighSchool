Gitの基本コマンド

最初に覚える7つのコマンド

# 1. リポジトリを初期化 $ git init # 2. 変更ファイルをステージに追加 $ git add index.html # 1ファイル $ git add . # 全ファイル # 3. コミット（変更を記録） $ git commit -m "CSSを追加" # 4. 現在の状態を確認 $ git status # 5. 変更履歴を表示 $ git log --oneline # 6. 変更内容の差分を表示 $ git diff

3つのエリア

作業ディレクトリ
実際にファイルを編集する場所
ステージ
git addで「次にコミットするファイル」を集める場所
リポジトリ
git commitで変更が永続的に記録される場所

ファイルを編集

コードを書く・修正する

git add

記録したいファイルをステージへ

git commit

変更を歴史として記録する
