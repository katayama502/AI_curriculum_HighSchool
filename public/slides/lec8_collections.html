リストと辞書の操作

# リスト操作
fruits = ["apple", "banana"]
fruits.append("cherry") # 追加
fruits.remove("banana") # 削除
len(fruits) # 長さ → 2

# リスト内包表記（強力！）
nums = [1, 2, 3, 4, 5]
evens = [x for x in nums if x % 2 == 0]
# → [2, 4]

# 辞書操作
student = {"name": "太郎", "score": 85}
student["grade"] = "A" # 追加
for key, val in student.items():
  print(f"{key}: {val}")
リストの主要メソッド
メソッド 説明
list.append(x) 末尾に要素を追加
list.remove(x) 指定した要素を削除
list.sort() 昇順に並び替え
len(list) 要素数を返す
⭐ リスト内包表記はPythonの最大の強み！
[式 for 変数 in リスト if 条件] の形で覚えよう
