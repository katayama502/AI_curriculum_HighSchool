SQLite：ファイル1つのデータベース

better-sqlite3 でCRUD操作

const Database = require('better-sqlite3');
const db = new Database('data.db');

// テーブル作成（初回のみ）
db.exec(`
  CREATE TABLE IF NOT EXISTS todos (
    id   INTEGER PRIMARY KEY AUTOINCREMENT,
    text TEXT NOT NULL,
    done INTEGER DEFAULT 0
  );`);

// データ追加
const insert = db.prepare('INSERT INTO todos (text) VALUES (?)');
insert.run('Reactを学ぶ');

// 全件取得
const all = db.prepare('SELECT * FROM todos');
console.log(all.all());

SQLiteの特徴

ファイル1つで完結
サーバー不要。data.db という1ファイルがデータベースになる。開発に最適。
better-sqlite3の強み
同期APIで書けるので非同期処理を気にせずシンプルに書ける。
本番では PostgreSQL へ
本番環境での大規模データはPostgreSQLへ移行。クエリ文法はほぼ同じ。
