技術スタックの選び方

🛠️ プロジェクト別おすすめスタック
📄 シンプルなWebページ
HTML + CSS + JavaScript
ポートフォリオ・紹介ページ
⚛️ インタラクティブなアプリ
React + Tailwind CSS
TodoApp・計算ツール・ゲーム
🗄️ データ保存が必要なアプリ
React + Express + SQLite
メモ帳・掲示板・スケジューラー
🤖 AI機能付きアプリ
React + Express + Claude API
AIチャット・自動要約ツール
🤔 技術選定の判断基準
データを保存する？
Yes → バックエンド必要
No → フロントのみでOK
ログイン機能が必要？
Yes → 認証システム必要
No → シンプルに作れる
AI機能を使う？
Yes → Claude API連携
No → 通常のWebアプリ
スマホ対応が必要？
Yes → Tailwindでレスポンシブ
No → デスクトップのみでOK
