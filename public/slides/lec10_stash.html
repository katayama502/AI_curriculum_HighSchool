git stash — 作業を一時退避する

こんな時に使う

困ったシナリオ
feature/loginブランチで作業中に、緊急バグ修正を頼まれた。でもコミットできる状態じゃない…どうする？
git stash
現在の変更を一時保存して、作業ディレクトリをきれいな状態に戻す
git stash list
保存したstashの一覧を確認する
git stash pop
最新のstashを取り出して作業を再開する
git stash drop
不要になったstashを削除する

stashの流れ

feature/loginで作業中
変更あり・コミット未完
git stash
変更を退避 → 作業ディレクトリがクリーンに
main に切り替えてバグ修正
git checkout main → 修正 → commit
feature/login に戻る
git checkout feature/login → git stash pop
ポイント：stashは「引き出しの中に一時保管」するイメージ。複数のstashをスタックで管理できます。
