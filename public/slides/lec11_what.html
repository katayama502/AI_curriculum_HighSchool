Claude Codeとは？

💬 Claude vs 🖥️ Claude Code
💬
Claude（Web）
ブラウザで会話するAI。質問・文章作成・学習に最適
🖥️
Claude Code
ターミナルで動くAI。コードを直接書いて実行する
最大の違い：Claude Codeはファイルを読み書きし、コマンドを実行し、プロジェクト全体を理解してコードを書ける！
🚀 Claude Codeの主な機能
📁
ファイルの読み書き・編集
プロジェクト全体を把握してコードを修正
⚡
コマンドの実行
npm install、git commit等を自動で実行
🐛
バグの自動修正
エラーを見て原因を特定・修正まで行う
🏗️
プロジェクト全体の構築
「〇〇を作って」で複数ファイルを生成
# インストール
npm install -g @anthropic-ai/claude-code
