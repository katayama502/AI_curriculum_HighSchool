変数とデータ型 — データを「箱」に入れる

変数の宣言と代入

変数 = 値を入れる「名前付きの箱」
// 変数の宣言（letを使う）
let name = "颯斗";
let age = 17;
let isStudent = true;

// 値を変更できる
age = 18; ← 書き換えOK

// 変更不可の定数
const PI = 3.14159;
基本的に const を使い、変更が必要な時だけ let を使う。var は古い書き方なので使わない。
