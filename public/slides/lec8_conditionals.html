条件分岐 — if / elif / else

インデント（字下げ）がブロックを作る

score = 75 if score >= 90: print("優秀！") elif score >= 70: print("合格") elif score >= 60: print("ギリギリ合格") else: print("不合格") # and / or も使える age = 17 has_id = True if age >= 18 and has_id: print("入場できます") elif age >= 16 or has_id: print("保護者同伴で入場可")

⚠️ インデント（スペース4個）が超重要！
{ } の代わりに字下げでブロックを表します。インデントがズレるとエラーになります。
