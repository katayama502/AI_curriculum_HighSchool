PythonでAPI取得（requests）

requestsライブラリで簡潔に書ける

import requests # GETリクエスト（天気API） params = { "latitude": 35.68, "longitude": 139.69, "current": "temperature_2m" } url = "https://api.open-meteo.com/v1/forecast" res = requests.get(url, params=params) # ステータスコード確認 print(res.status_code) # 200 = 成功 # JSONデータを辞書として取得 data = res.json() temp = data["current"]["temperature_2m"] print(f"東京の気温: {temp}℃") # POSTリクエスト（データを送る） res2 = requests.post(api_url, headers={"Content-Type": "application/json"}, json={"message": "こんにちは"} )

JSとPythonの比較

JavaScript (fetch)
await fetch(url)
→ await res.json()
（async/awaitが必要）
Python (requests)
requests.get(url)
→ res.json()
（同期的に書ける！）

💡 requestsの便利な点：
paramsに辞書を渡すだけでクエリ文字列を自動生成。jsonでPOSTボディも簡単に送れる。同期的に書けるのでコードが読みやすい。
