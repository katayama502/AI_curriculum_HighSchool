Claude Code 環境構築

🔧 セットアップ手順
1
Node.jsをインストール
nodejs.org からLTS版をダウンロード
2
Claude Codeをインストール
npm install -g @anthropic-ai/claude-code
3
APIキーを設定
export ANTHROPIC_API_KEY=sk-ant-...
4
プロジェクトフォルダで起動
cd my-project && claude
5
最初の指示を入力
「Hello World を表示するHTMLを作って」
💻 ターミナルの様子
$ claude
✓ Claude Code v1.x.x
API key: ****...****

Claude> Hello Worldを表示するHTMLを作って

● index.html を作成します...
