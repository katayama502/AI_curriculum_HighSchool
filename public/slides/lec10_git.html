Gitとは？バージョン管理の基本

Gitとは：
コードの変更履歴を記録・管理するツール。「いつ・誰が・何を変更したか」を追跡できる。
📚 ゲームのセーブデータで例えると
💾
commit = セーブポイントを作成
「ここまでの進捗を保存」
⏪
checkout = 過去のセーブに戻る
「バグが出たら以前の状態に戻せる」
🌿
branch = 別ルートで試す
「本編を壊さず新機能を試せる」
基本コマンド一覧
コマンド 説明
git init Gitリポジトリを初期化
git status 変更状態を確認
git add . 変更をステージングに追加
git commit -m 変更を記録（メッセージ付き）
git log コミット履歴を表示
💡 Claudeに「このエラーのgitコマンドを教えて」と聞くと即解決！
