# 📚 読書記録アプリ

読んだ本を記録・管理できるWebアプリです。
[デモを見る](https://your-app.vercel.app)

## 機能
- ユーザー認証（JWT）
- 本の CRUD 管理
- レスポンシブデザイン

## 技術スタック
Frontend: React + Vite + Tailwind CSS
Backend: Express.js + SQLite
Deploy: Vercel + Railway
