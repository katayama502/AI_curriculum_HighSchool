CSSアニメーションでUXを向上

よく使うアニメーション

適切なアニメーションはUIをリッチにし、ユーザーの操作感を向上させます。

Hover浮き上がり
カードやボタンのhoverで使用。立体感を演出
フェードイン/アウト
モーダルや通知の表示・非表示に使用
スケール（拡大縮小）
ボタンクリック時やアイコンのhoverに使用
アニメーションはやりすぎNG！ duration は 200〜400ms が心地よい

CSSの書き方

/* Hover浮き上がり */
.card {
transition: transform 0.2s, box-shadow 0.2s;
}
.card:hover {
transform: translateY(-4px);
box-shadow: 0 8px 24px rgba(0,0,0,0.12);
}

/* フェードイン */
@keyframes fadeIn {
from { opacity: 0; transform: translateY(10px); }
to { opacity: 1; transform: translateY(0); }
}
.modal {
animation: fadeIn 0.3s ease-out;
}

/* ボタンクリック */
.btn:active {
transform: scale(0.96);
}
