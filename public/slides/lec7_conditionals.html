条件分岐 — if/else で判断を作る

基本の if / else if / else

const score = 75;

if (score >= 90) {
  console.log("優秀！");
} else if (score >= 70) {
  console.log("合格");
} else {
  console.log("再挑戦");
}

// 出力: "合格"
比較演算子
===等しい !==等しくない >より大きい <より小さい >=以上 &&かつ ||または
