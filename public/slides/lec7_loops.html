ループ — 繰り返し処理を使いこなす

for ループの基本

// 0から4まで5回繰り返す
for (let i = 0; i < 5; i++) {
  console.log(i); // 0,1,2,3,4
}

// 配列を順番に処理
const fruits = ["りんご", "バナナ", "みかん"];
fruits.forEach((fruit) => {
  console.log(fruit);
});

// 出力: りんご / バナナ / みかん

3種のループを使い分ける

for ループ
回数が決まっている時。インデックスが必要な時。
for (let i = 0; i < 10; i++) { ... }
for...of ループ
配列の各要素を処理する時（最もシンプル）
for (const item of array) { ... }
while ループ
条件が真の間ずっと繰り返す時
while (count < 10) { count++; }
🎯 最もよく使うのは forEach / for...of
配列の全要素を処理する時
