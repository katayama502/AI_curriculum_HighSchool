JavaScript チートシート

📦 変数・配列

// 変数
let name = "太郎";
const PI = 3.14;

// 配列
const arr = ["a", "b", "c"];
arr.push("d");     // 追加
arr.forEach(x => console.log(x));
arr.map(x => x + "!"); // 変換
arr.filter(x => x !== "a"); // 絞込

🌐 fetch / async

// データ取得
async function getData() {
    const res = await fetch(url);
    const data = await res.json();
    // dataを使って画面更新
    document.getElementById("out")
        .textContent = data.name;
}

この4つをマスターすれば、インタラクティブなWebページが自分で作れる！わからないときはClaudeに聞きながら進もう。
