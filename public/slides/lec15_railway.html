Railway：Express サーバーをデプロイ

デプロイ手順

1
package.json に start スクリプトを追加
"scripts": { "start": "node index.js" }
2
railway.app でGitHub連携
「New Project」→「Deploy from GitHub repo」でリポジトリを選択。
3
環境変数を設定
Variables タブに .env の内容を入力。PORTは自動設定されるので不要。
4
Expressで PORT を動的に取得する
app.listen(process.env.PORT || 3000);

注意点

SQLiteは本番では使えない
RailwayはファイルシステムをリセットするためSQLiteのデータが消える。本番ではRailwayのPostgreSQLアドオンを使う。
CORSを本番URLに変更
origin を process.env.FRONTEND_URL にして、VercelのURLを環境変数に入れよう。
無料プランの制限
Railwayの無料枠は月$5相当のクレジット。学習用であれば十分に使える。
