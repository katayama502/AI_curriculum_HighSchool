複数ファイルへの変更例

「ダークモードを追加して」と頼むと Claude Codeは HTML・CSS・JS を一度に更新してくれる！
📄
index.html
→ data-theme属性を追加
同時に変更
🎨
style.css
→ [data-theme="dark"]のスタイルを追加
同時に変更
⚡
script.js
→ トグルボタンのイベントを追加

効果的な依頼の仕方

// ✓ 良い例：影響範囲を意識した依頼
ダークモード切り替えボタンを追加してください。

変更が必要なファイル：
- index.html：ヘッダーにボタンを追加
- style.css：ダークテーマの色定義
- script.js：クリックでdata-theme切り替え

各ファイルの変更点も教えてください。
変更前にバックアップする習慣
複数ファイルを変更する前に git commit しておくと、問題が起きたときに git checkout . で元に戻せる。
変更内容を確認する
Claude Codeが変更を提案したら「/review」で差分を確認してから承認する。盲目的にyesを押さない！
