スラッシュコマンドを使いこなす

主なスラッシュコマンド

/help
使えるコマンド一覧を表示する
/clear
会話履歴をリセット。新しい話題を始めるときに使う
/review
変更したコードをレビューしてもらう
/init
プロジェクトのCLAUDE.mdを生成する
/compact
長い会話を要約してトークンを節約する
/cost
今のセッションのAPI使用コストを確認
/bug
バグ報告のフォーマットで問題を整理する
/exit
Claude Codeを終了する

実際の使用例

Claude Code
claude> /help
Available commands:
/clear /review /init ...

claude> /init
✓ Created CLAUDE.md
プロジェクトの説明が自動生成

claude> index.htmlを修正して
✓ Edited index.html

claude> /review
変更点のレビュー結果が表示...

claude> /clear
✓ 会話リセット完了
コツ：長い作業の後は /compact でコンテキストを圧縮すると、レスポンスが速くなる！
