Webの仕組み — ブラウザとサーバーの会話

3つの主役

🌐
HTML
骨格（構造）を担当
Webページの「中身と構造」を定義。見出し・段落・リンク・画像の配置
🎨
CSS
見た目（デザイン）を担当
色・フォント・余白・レイアウトなど「スタイル」をすべて制御
⚡
JavaScript
動き（動作）を担当
ボタンのクリック反応・アニメーション・動的なコンテンツ変更

ページが表示されるまでの流れ

URLを入力してからページが見えるまで
① ユーザーがURLを入力
② ブラウザがサーバーにリクエスト送信
③ サーバーがHTMLファイルを返送
④ ブラウザが解析して画面に表示
料理に例えると：HTMLが食材、CSSが盛り付け・味付け、JavaScriptが食べ方の演出。
