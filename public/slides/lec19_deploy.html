本番デプロイの手順

🚀 デプロイ前の最終確認
1
環境変数を設定する
.envファイルをVercelの設定画面に入力
2
ビルドが通ることを確認
npm run build でエラーがないか確認
3
GitHubにpushする
git add . && git commit && git push
4
Vercelで自動デプロイを確認
デプロイログを確認してエラーがないか確認
5
本番URLで動作確認
全機能をもう一度テストする
💻 デプロイをClaudeに手伝ってもらう
Claude> Vercelへのデプロイで
エラーが出ています：
Error: Missing env variable
DATABASE_URL

● エラーを分析中...
原因：Vercelに環境変数が
設定されていません

解決方法：
1. Vercelダッシュボードを開く
2. Settings → Environment Variables
3. DATABASE_URL を追加する
4. Redeploy を実行する
✓ これで解決するはずです！
