関数とモジュールの活用

# デフォルト引数付き関数
def greet(name, lang="ja"):
  if lang == "ja":
    return f"こんにちは、{name}！"
  else:
    return f"Hello, {name}!"

# モジュールのインポート
import random
import datetime

# ランダムな数を生成
num = random.randint(1, 100)

# 今日の日付を取得
today = datetime.date.today()
print(f"今日は{today}です")
よく使う標準モジュール
random
乱数生成・リストのシャッフル
datetime
日付・時刻の取得と計算
math
数学関数（sqrt, sin, cos等）
json
JSON形式のデータを読み書き
