APIとは？

Application Programming Interface

🍽️ レストランのメニュー表のようなもの

お客さん（あなたのアプリ）がメニュー（API）を見て注文（リクエスト）すると、キッチン（サービス）が料理（データ）を出してくれる。キッチンの中がどうなっているか知らなくていい。
