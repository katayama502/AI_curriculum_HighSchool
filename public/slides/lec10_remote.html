インターネット上のサーバー

# GitHubにリポジトリを登録 $ git remote add origin https://github.com/user/repo.git # ローカルの変更をGitHubに送る $ git push origin main # GitHubの変更をローカルに取得 $ git pull origin main # 他人のリポジトリをコピーして取得 $ git clone https://github.com/user/repo.git

初めてのGitHub連携手順

  • 1
    GitHub.comでリポジトリを作成
    「New repository」→名前を入力→Create
  • 2
    ローカルでgit init
    プロジェクトフォルダで初期化する
  • 3
    git remote add origin [URL]
    GitHubのリポジトリURLを登録する
  • 4
    git add . → git commit → git push
    ファイルをコミットしてGitHubに送る
  • 5
    GitHub上でコードを確認
    ブラウザでリポジトリが公開される
