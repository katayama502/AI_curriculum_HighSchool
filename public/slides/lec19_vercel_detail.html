Vercelでフロントエンドをデプロイ

デプロイ手順（5分で完了）

1
GitHubにコードをpush
git push origin main でリモートへ反映
2
vercel.com でログイン
GitHubアカウントでサインイン→「New Project」
3
リポジトリを選択してImport
フロントエンドのリポジトリを選ぶ（client/）
4
環境変数を設定
VITE_API_URL = https://your-api.railway.app
Settings → Environment Variables
5
「Deploy」をクリック
1〜2分で https://your-app.vercel.app が完成！

Vercelの便利な機能

🔄
自動デプロイ
mainブランチにpushするだけで自動でデプロイが走る
👀
プレビューURL
PRごとにプレビュー環境が自動生成される
🌍
グローバルCDN
世界中のサーバーから配信。日本からも爆速
🔒
HTTPS自動設定
SSL証明書が無料で自動設定される
// React側のAPI URLを環境変数で切り替え
// .env.local（開発用）
VITE_API_URL=http://localhost:3000
// Vercel（本番用）
VITE_API_URL=https://your-api.railway.app
