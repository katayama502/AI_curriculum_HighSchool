パーミッションと安全な使い方

Claude Codeができること

📖
ファイルの読み取り
プロジェクト内のファイルを全て読むことができます。実行前に確認は不要。
✏️
ファイルの書き込み・編集
ファイルを変更する前に「許可しますか？」と聞いてきます（確認が必要）。
⚡
コマンドの実行
npm install や git コマンドを実行する前に必ず確認を求めます。
このような確認が出ます
「index.htmlを編集してもいいですか？」
「npm install を実行してもいいですか？」
「このファイルを削除してもいいですか？」
