フォーム：Controlled Component

Reactでフォームを制御する

import { useState } from "react";

function SearchForm() {
  const [city, setCity] = useState("");

  function handleSubmit(e) {
    e.preventDefault(); // ページリロードを防ぐ
    console.log("検索する都市:", city);
  }

  return (
    <form onSubmit={handleSubmit}>
      <input
        value={city}
        onChange={e => setCity(e.target.value)}
        placeholder="都市名を入力"
      />
      <button type="submit">検索</button>
    </form>
  );
}

ポイント

1
value と state を同期
inputの value をstateで管理することで、Reactが入力内容を常に把握できます。
2
onChange で state 更新
e.target.value が入力された文字列。setCity でstateを更新します。
3
onSubmit で送信処理
e.preventDefault() を忘れずに！ページリロードを防ぎます。
