useEffect — 副作用とAPIデータ取得

useEffectの基本構造

基本形 — コンポーネント起動時に実行
import { useState, useEffect } from 'react';

function WeatherApp() {
const [temp, setTemp] = useState(null);

// 第2引数の[]は「初回のみ実行」を意味
useEffect(() => {
async function fetchData() {
const res = await fetch(API_URL);
const data = await res.json();
setTemp(data.current.temperature_2m);
}
fetchData();
}, []); // ← 空の依存配列

return <p>{temp ? `${temp}°C` : "読み込み中..."}</p>;
}

よく使うHooks一覧

useState
状態（数値・文字列・配列など）を管理する。変わるとUIが自動更新される。
useEffect
API呼び出し・タイマー・イベントリスナーなど「副作用」を実行する。
useRef
DOM要素への参照を保持する。input要素にフォーカスする時などに使う。
useContext
深くネストしたコンポーネントへPropsなしでデータを渡す（グローバル状態）。
