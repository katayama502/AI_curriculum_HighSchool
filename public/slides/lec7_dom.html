DOM操作でページを動かす

// HTML要素を取得
const btn = document.querySelector("#myBtn");
const msg = document.querySelector("#message");

// クリックイベントを設定
btn.addEventListener("click", () => {
  msg.innerHTML = "ボタンが押された！";
  msg.style.color = "#339AF0";
});

// 要素を追加
const newItem = document.createElement("li");
newItem.textContent = "新しいアイテム";
list.appendChild(newItem);
主要なDOMメソッド
querySelector()
CSS セレクタで要素を取得
innerHTML
要素の中身のHTMLを変更
addEventListener()
クリック等のイベントを監視
createElement()
新しいHTML要素を作成
