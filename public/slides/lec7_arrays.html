配列（Array）

複数のデータをまとめて管理

// 配列の作り方 const fruits = ["りんご", "バナナ", "みかん"]; // インデックス（0始まり）でアクセス console.log(fruits[0]); // "りんご" console.log(fruits[2]); // "みかん" // 要素数を確認 console.log(fruits.length); // 3 // forループで全要素を処理 for (let i = 0; i < fruits.length; i++) { console.log(fruits[i]); }

💡 ポイント：配列の番号は0から始まる！最初の要素は[0]、2番目は[1]。これはプログラミングの世界の共通ルールです。

よく使うメソッド一覧

メソッド役割例
.push()末尾に追加fruits.push("ぶどう")
.pop()末尾を削除fruits.pop()
.shift()先頭を削除fruits.shift()
.unshift()先頭に追加fruits.unshift("メロン")
.indexOf()位置を検索fruits.indexOf("バナナ")
.includes()含むか確認fruits.includes("りんご")
.forEach()全要素に処理fruits.forEach(f => ...)
.map()変換した新配列fruits.map(f => f+"🍎")
.filter()条件で絞り込みnums.filter(n => n > 3)
