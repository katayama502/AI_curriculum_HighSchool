Vercel：React アプリを無料でデプロイ

デプロイ手順

1
vercel.com でアカウント作成
GitHubアカウントでサインアップするだけ。無料プランで十分。
2
「Import Project」でGitHubリポジトリを選択
Vite + React のプロジェクトを選ぶと自動検出してくれる。
3
環境変数を設定してデプロイ
Settings → Environment Variables に .env の内容を入力。
# または CLI でも可能
$ npm install -g vercel
$ vercel
✓ Deployed: https://my-app.vercel.app

Vercelの特徴

push = 自動デプロイ
GitHubにpushするたびに自動でビルド＆デプロイ。mainブランチのコードが常に本番に反映される。
プレビューデプロイ
プルリクエストごとにプレビューURLが発行される。確認してからmainにマージできる。
独自ドメインも無料
xxx.vercel.app のURLに加え、購入したドメインをVercelに向けることもできる。
Vercel はフロントエンドのみ。APIサーバーは別途Railwayなどを使う。
