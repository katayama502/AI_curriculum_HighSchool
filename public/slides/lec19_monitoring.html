アクセス解析とモニタリング

公開後は「誰が来ているか」「エラーが起きていないか」を監視することが大切です。

📊
Google Analytics
無料
訪問者数・ページビュー・滞在時間・流入元を計測。Googleアカウントで使える
⚡
Vercel Analytics
無料プランあり
Core Web Vitals（ページ速度）を計測。Vercelダッシュボードで確認できる
🚨
Sentry
無料プランあり
本番環境のエラーをリアルタイムで通知。エラーのスタックトレースも確認できる
💓
UptimeRobot
無料
5分ごとにAPIへアクセス確認。ダウンしたらメールで通知してくれる
📝
Railway / Vercel ログ
無料
console.log の出力を確認できる。エラーの原因調査に必須
🔍
Lighthouse CI
無料
パフォーマンス・アクセシビリティ・SEOを継続的に計測。GitHubと連携可能
LCP
最大描画（2.5s以内）
FID
応答速度（100ms以内）
CLS
レイアウトずれ（0.1以内）
これらCore Web VitalsはGoogleの検索順位にも影響する重要な指標！
