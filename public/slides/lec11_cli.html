Claude Code の使い方・コマンド

主要コマンド一覧
コマンド 説明
/help 使い方ガイドを表示
/clear 会話履歴をリセット
/exit Claude Codeを終了
Ctrl+C 実行中の処理を中断
claude . 現在のフォルダで起動
✨ 効果的な指示の出し方
🎯
具体的に指示する
「何を」「どのように」「どんな形で」を明確に
📋
コンテキストを与える
「このプロジェクトはReactで作っている」など
🔄
段階的に依頼する
一度に全部頼まず、少しずつ確認しながら進める
❌ 悪い例
「Webサイトを作って」
✅ 良い例
「ポートフォリオサイトのHTMLを作って。ヘッダー・自己紹介・作品一覧の3セクション」
