関数の定義と活用

def で自分だけの関数を作る

# 基本の関数定義 def greet(name, greeting="こんにちは"): """あいさつを返す関数（docstring）""" return f"{greeting}、{name}さん！" # 呼び出し（デフォルト引数使用） print(greet("太郎")) # → こんにちは、太郎さん！ # 引数を指定して呼び出し print(greet("花子", "おはよう")) # → おはよう、花子さん！ # 複数の値を返す def calc_stats(nums): return min(nums), max(nums), sum(nums)/len(nums) lo, hi, avg = calc_stats([60, 75, 90, 85])

関数の構造

def
関数の定義を開始するキーワード
関数名
英小文字+アンダースコアで命名
引数
()内に受け取る値を定義。デフォルト値も設定可
docstring
関数の説明文。"""…"""で囲む（省略可）
return
呼び出し元に値を返す。なければNone

💡 関数を使うメリット
同じ処理を何度も書かなくていい（DRY原則）。バグが出たとき1箇所直すだけでOK。コードが読みやすくなる。
