デプロイ前の最終チェック

コードチェックリスト

✅
.env は .gitignore に入っているか
絶対にGitHubにpushしない。secrets漏洩の最大原因
✅
console.log のデバッグ出力を削除したか
本番環境ではパスワードなど機密情報がログに残らないよう注意
✅
npm run build でビルドが通るか
TypeScriptエラーやビルドエラーが出ないことを確認
✅
node_modules は .gitignore に入っているか
package.json があればnpm installで再生成できる
✅
package.json に start スクリプトがあるか
Railway等が "npm start" を実行するので必須
絶対禁止： API_KEY や JWT_SECRET を直接コードに書かない！

必要な設定ファイル

// package.json の scripts
{
"scripts": {
"start": "node server.js",
"dev": "nodemon server.js",
"build": "vite build"
}
}

// .gitignore
node_modules/
.env
dist/
*.db

// PORT を環境変数から取得（必須）
const PORT = process.env.PORT || 3000;
app.listen(PORT, () => {
console.log(`起動: port ${PORT}`);
});
