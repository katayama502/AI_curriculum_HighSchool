HTTPメソッドとステータスコード

4つの主要メソッド（CRUD）

GET

データを取得する

URLでデータを読み取る。何も変更しない。ブラウザでURLを開く動作はGET。

POST

データを送信・作成する

フォーム送信、新規データ作成。リクエストの本文にデータを含めて送る。

PUT

データを更新する

既存データの上書き・更新。全フィールドを送る必要がある。

DELETE

データを削除する

指定したリソースを削除する。

ステータスコード（サーバーの返答）

コード意味
200 OK成功！正常に処理された
201 Created作成成功（POSTに多い）
400 Bad Requestリクエストが不正（パラメータ不足など）
401 Unauthorized認証が必要（APIキー未設定）
403 Forbiddenアクセス権限がない
404 Not FoundURLが存在しない
429 Too Manyリクエスト数上限を超えた
500 Server Errorサーバー側の問題
