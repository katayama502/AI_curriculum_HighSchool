メディアクエリの書き方

/* PC向けスタイル（デフォルト） */
.container {
  display: flex;
  gap: 40px;
}

/* スマホ向け（画面幅600px以下） */
@media (max-width: 600px) {
  .container {
    flex-direction: column;
    gap: 16px;
  }
}
💡 Claudeに「このHTMLをスマホ対応にして」と頼むと、メディアクエリを自動で追加してくれる。自分で書く必要はない！
