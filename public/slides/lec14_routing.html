ルーティング：URLとメソッドの組み合わせ

Express ルーティングの基本

const express = require('express');
const app = express();

app.use(express.json()); // JSONを受け取る設定

// GET: データを取得する
app.get('/api/users', (req, res) => {
  res.json([{ id: 1, name: '太郎' }]);
});

// POST: 新しいデータを作成する
app.post('/api/users', (req, res) => {
  const { name } = req.body;
  res.json({ id: 2, name });
});

// URLパラメータ (:id)
app.get('/api/users/:id', (req, res) => {
  res.json({ id: req.params.id });
});

app.listen(3000, () => console.log('起動中!'));

HTTPメソッド

GET
データを取得する（読み取り専用）
POST
新しいデータを作成する
PUT
既存のデータを更新する
DELETE
データを削除する
この4つの組み合わせを CRUD（クラッド）と呼ぶ。Webアプリの基本操作！
