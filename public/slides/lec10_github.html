🔄 GitとGitHubの関係
💻
ローカルリポジトリ（自分のPC）
git add → git commit で変更を記録
↕ git push / git pull
☁️
リモートリポジトリ（GitHub）
クラウドにコードを保存・公開
🌐 GitHub Pages
GitHubのリポジトリから
無料でWebサイトを公開できる！
username.github.io/repo-name でアクセス可能
GitHub連携コマンド
git remote add
GitHubリポジトリを登録
git push origin
ローカルの変更をGitHubに送信
git pull origin
GitHubの変更をローカルに取得
git clone
GitHubのリポジトリをコピー
git branch
ブランチの作成・切り替え
