GitHub Pages — 作品をWebで公開する

公開までの手順

1
GitHubにリポジトリを作成してpushする
index.htmlがルートディレクトリにあることを確認
2
Settings → Pages を開く
リポジトリページ上部の「Settings」タブをクリック
3
Source を「Deploy from a branch」に設定
Branch: main / folder: / (root) を選択して Save
4
数分待つとURLが発行される
自動でサイトがビルドされ、世界に公開される！
公開URLの形式
https://ユーザー名.github.io/リポジトリ名/

GitHub Pages の特徴

🆓
完全無料で使える
静的サイト（HTML/CSS/JS）のホスティングが無料。ポートフォリオ公開に最適！
🔄
pushするだけで自動更新
git push するたびに自動でサイトが更新される。FTPアップロード不要。
⚠️
静的サイトのみ対応
サーバーサイド（Python/Node.js）は動かない。HTML/CSS/JSのみ。APIキーを書いたまま公開しないよう注意！
