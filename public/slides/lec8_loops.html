ループ処理 — for / while

繰り返しを自動化する

# range()で回数指定ループ for i in range(5): print(f"{i}回目") # 0〜4 # リストの要素を順番に処理 fruits = ["りんご", "バナナ", "みかん"] for fruit in fruits: print(fruit + "が好き") # enumerate()で番号付き for i, fruit in enumerate(fruits): print(f"{i+1}位: {fruit}") # while：条件が真の間ループ count = 0 while count < 3: print(f"カウント: {count}") count += 1

ループの制御

break

条件が満たされたらループを即終了。「見つかったら止まる」探索に使う。

continue

その回をスキップして次の繰り返しへ。「特定の要素を除外」したいときに使う。

range(start, stop, step)

range(0, 10, 2) → 0, 2, 4, 6, 8
range(10, 0, -1) → 10〜1（カウントダウン）

リスト内包表記

[x*2 for x in range(5)]
→ [0, 2, 4, 6, 8]
1行でリストを生成できる省略記法。
