JWTとは？認証の仕組み

1
ユーザーがメール+パスワードでログイン
2
サーバーがパスワードを検証（bcryptで比較）
3
サーバーがJWTトークンを生成して返す
4
ブラウザはトークンをlocalStorageに保存
5
以後のAPIリクエストにトークンを添付
6
サーバーがトークンを検証→本人確認OK

Express実装例

// POST /api/auth/login
router.post('/login', (req, res) => {
const { email, password } = req.body;
const user = db.prepare(
'SELECT * FROM users WHERE email=?'
).get(email);

if (!user) return res.status(401)
.json({ error: 'ユーザーが見つかりません' });

const ok = bcrypt.compareSync(password, user.hash);
if (!ok) return res.status(401)
.json({ error: 'パスワードが違います' });

const token = jwt.sign(
{ userId: user.id },
process.env.JWT_SECRET,
{ expiresIn: '7d' }
);
res.json({ token });
});
