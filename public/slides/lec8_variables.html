変数とデータ型

型宣言不要！シンプルに書ける

# 変数への代入（型を書かなくていい！） name = "田中太郎" age = 17 height = 168.5 is_student = True # 型を確認する print(type(name)) # <class 'str'> print(type(age)) # <class 'int'> # 文字列 + 数値を組み合わせる print(f"私は{age}歳です") # f文字列 print("名前：" + name) # 連結

JSとの違い

JavaScriptは let / const が必要だが、Pythonは変数名だけでOK。
インデントがブロックを表す（{ }不要）。
