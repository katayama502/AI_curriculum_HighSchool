Lighthouse の使い方
Chrome の DevTools（F12）を開き、「Lighthouse」タブを選択。「Analyze page load」ボタンをクリックするだけで全スコアが計測される。
目標：Performance 90以上、Accessibility 90以上、SEO 80以上を目指そう！

改善ポイント

画像を最適化
JPG・PNG の代わりに WebP形式を使う。サイズを指定して不要な読み込みを減らす。
コードを圧縮
Viteは npm run build で自動的に圧縮・最適化される。本番はビルド版を使うこと。
アクセシビリティ
img タグに alt を追加、button に aria-label を追加するだけでスコアが上がる。
SEO 基本設定
title・meta description タグを設定。OGP画像を追加するとSNSシェアも映える。
