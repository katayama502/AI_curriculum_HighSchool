DOMを操作する

DOM＝HTMLをJSで操作するための仕組み

document
  └ <html>
      ├ <head>
      └ <body>
          ├ <h1 id="title">見出し
          └ <p class="text">本文
// 要素を取得 const title = document.getElementById("title"); const texts = document.querySelectorAll(".text"); // テキストを変更 title.textContent = "新しい見出し"; // スタイルを変更 title.style.color = "#339AF0"; // HTMLを追加 title.innerHTML = "<strong>太字</strong>";

取得メソッド

getElementById("id") — IDで1つ取得
querySelector(".class") — CSSセレクタで1つ
querySelectorAll("tag") — 複数まとめて取得

変更プロパティ

.textContent — テキストを変更
.innerHTML — HTMLごと変更
.style.プロパティ — CSSを直接変更
.classList.add("名前") — クラスを追加

活用例

ボタンを押したらテキストが変わる
入力した名前を画面に表示する
正解/不正解で背景色を変える
