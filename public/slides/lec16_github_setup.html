GitHubでプロジェクトを管理しよう

リポジトリ初期設定手順

1
GitHubでリポジトリを作成
github.com → New repository → 名前を付けてCreate
2
ローカルでプロジェクトを初期化
git init && git add . && git commit -m "init"
3
リモートに紐付けてプッシュ
git remote add origin [URL]
git push -u origin main
4
.gitignore を設定（重要！）
node_modules/ と .env は必ずIgnore対象に
コミットは「動く状態」ごとにこまめに行う習慣をつけよう
ブランチ戦略
# メインブランチ
main ← 常に動く状態を保つ

# 開発ブランチ
develop ← 開発の統合先

# 機能ブランチ
feature/auth
feature/book-list
feature/deploy

# 作業の流れ
① feature/ で作業
② develop にPull Request
③ 確認後 main にマージ
