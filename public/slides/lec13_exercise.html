演習課題 — React 基礎

課題A プロフィールカードコンポーネント
名前・年齢・趣味をpropsで受け取るカード型コンポーネントを作成し、3人分のデータをmapで表示しよう。
課題B カウンターアプリ
useStateを使って「+」「−」ボタンで数値を増減させるカウンターを作成しよう。0以下にならないようにしよう。
課題C 天気API表示
useEffectでOpen-Meteo APIから東京の現在気温を取得し、ローディング・エラー処理も含めて表示しよう。
★ 発展 都市検索天気アプリ
フォームで都市名を入力して検索できる天気アプリを完成させよう。コンポーネントに分割して実装しよう。

取り組み方

1
npm create vite@latest でプロジェクト作成
2
Claude Codeに「コンポーネントの役割」を伝えて生成
3
npm run dev でブラウザ確認しながら修正
4
完成したら git commit で保存
Claude Codeに「〇〇のReactコンポーネントを作って」と伝えるだけで下地が完成！
