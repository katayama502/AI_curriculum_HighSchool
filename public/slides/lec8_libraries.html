ライブラリの活用

importで機能を追加する

# 標準ライブラリ（インストール不要） import math import random import datetime # 使い方 print(math.sqrt(16)) # 4.0 平方根 print(math.pi) # 3.1415... n = random.randint(1, 6) # さいころ items = ["A", "B", "C"] random.shuffle(items) # シャッフル today = datetime.date.today() print(f"今日は{today}") # 別名でimport（よく使う） import numpy as np import pandas as pd
$ pip install numpy pandas requests

有名な外部ライブラリ

numpy

数値計算・行列演算。科学・AIの基盤。

pandas

表形式のデータ分析。Excelの上位互換。

matplotlib

グラフ・チャートを描画する可視化ライブラリ。

requests

WebからデータをHTTPで取得する。

flask

軽量WebフレームワークでAPIを作れる。

anthropic

Claude APIを呼び出す公式ライブラリ。
