Expressのエラー処理パターン

// バリデーション（入力チェック）
app.post('/api/todos', (req, res) => {
  const { text } = req.body;

  if (!text || text.trim() === '') {
    return res.status(400).json({
      error: 'テキストは必須です'
    });
  }

  try {
    const result = insert.run(text);
    res.status(201).json({ id: result.lastInsertRowid });
  } catch (err) {
    res.status(500).json({ error: 'サーバーエラー' });
  }
});

// 404 ハンドラー（最後に定義）
app.use((req, res) => {
  res.status(404).json({ error: '見つかりません' });
});

HTTPステータスコード

200
OK — 成功（GET・PUT）
201
Created — 作成成功（POST）
400
Bad Request — 入力が不正
404
Not Found — 見つからない
500
Server Error — サーバー側の問題
