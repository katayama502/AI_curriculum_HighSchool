Claude APIを使う

Pythonで自分のアプリにAIを組み込む

import anthropic # クライアントを初期化（APIキーは環境変数から自動読み込み） client = anthropic.Anthropic() # メッセージを送ってAIの返答を受け取る message = client.messages.create( model="claude-opus-4-6", max_tokens=1024, messages=[ { "role": "user", "content": "量子コンピュータを高校生向けに説明して" } ] ) # 返答を取り出す print(message.content[0].text)

💡 pip install anthropic でインストール。APIキーはANTHROPIC_API_KEYという名前で環境変数に設定する。

主要パラメータ

model
使うモデル名。claude-opus-4-6、claude-sonnet-4-6など
max_tokens
返答の最大文字数（トークン数）。長すぎると課金増加
messages
role（user/assistant）とcontentのリスト。会話履歴も含められる
system
AIの役割・制約を設定するシステムプロンプト
temperature
0〜1でランダム性を調整。0=確実・1=創造的
