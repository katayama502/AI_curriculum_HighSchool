System Prompt の活用

System Promptとは：
会話の「前提条件」をAIに設定するための特別な指示。ユーザーの質問より前に処理される。
活用シーン
🎭 AIのキャラクター・役割を固定する
📏 回答の形式・長さを統一する
🌐 使用言語・専門レベルを設定する
🔒 特定のトピックに絞り込む
📝 System Prompt 例：勉強サポートBot
# あなたの役割
あなたは高校生向けの
勉強サポートAIです。

# 回答ルール
- 中学〜高校レベルで説明
- 必ず具体例を1つ含める
- 300字以内で回答する
- 励ましの言葉で締める
→ 効果
毎回同じ指示をしなくても、
常に一貫したスタイルで回答してくれる！
