CSSボックスモデル — 余白を使いこなす

すべての要素は「箱」でできている

MARGIN（外側の余白）
BORDER（枠線）
PADDING（内側の余白）
CONTENT
テキスト・画像など
margin 他要素との距離
border 要素の境界線
padding 内容との距離
content 実際の内容

実際のCSSコード

.card {
  width: 300px;
  padding: 20px; ← 内側余白
  margin: 16px 0; ← 外側余白
  border: 2px solid #dee2e6;
  border-radius: 8px; ← 角丸
  background-color: white;
}
