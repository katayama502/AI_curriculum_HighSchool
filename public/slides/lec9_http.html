HTTPメソッドとJSON

HTTPメソッドの種類
メソッド 操作 用途
GET Read データを取得する
POST Create 新しいデータを送信する
PUT Update 既存データを更新する
DELETE Delete データを削除する
💡 Webアプリでは主に GET（取得）と POST（送信）を使う
JSONデータの形式
{
  "name": "山田太郎",
  "age": 16,
  "subjects": ["数学", "英語"],
  "isStudent": true
}
🐍 Pythonでの使い方
import requests

# GETリクエスト
res = requests.get("https://api.example.com/data")
data = res.json() # JSON→辞書に変換
print(data["name"])
