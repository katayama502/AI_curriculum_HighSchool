JSX — JavaScriptの中にHTMLを書く

JSXとは？

JSXはJavaScriptの拡張構文。HTML風の見た目でUIを書けるが、裏側はJavaScriptに変換される。Reactの中心的な書き方。
基本的なJSXの書き方
// Reactコンポーネント
function Greeting() {
const name = "颯斗";
return (
<div className="card">
<h1>こんにちは、{name}さん！</h1>
<p>今日もプログラミングを頑張ろう</p>
</div>
);
}
{ } で式を埋め込む
const score = 95;
return <p>点数: {score}点</p>;
// → <p>点数: 95点</p>

// 条件分岐も書ける
<p>{score >= 60 ? "合格" : "不合格"}</p>

JSXの重要なルール

classではなくclassNameを使う
<div className="container">
必ず1つのルート要素で囲む
return (
<div>
<h1>...</h1>
<p>...</p>
</div>
);
自己閉じタグが必要
<img src="photo.png" />
<input type="text" />
コンポーネント名は大文字始まり
function MyButton() { ... } // ✓
function mybutton() { ... } // ✗
スタイルはオブジェクトで指定
<div style={{ color: 'red', fontSize: '20px' }}>
