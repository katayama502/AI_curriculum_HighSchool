演習課題 — Git・GitHub

TASK A — 基礎

最初のコミット

新しいフォルダを作成してgit initし、HTMLファイルを作成してgit add → git commitの一連の流れを実践しよう。

  1. mkdir my-project && cd my-project
  2. git init
  3. index.htmlを作成してコードを書く
  4. git add . → git commit -m "初回コミット"
TASK B — 応用

GitHubに公開する

Task Aで作ったリポジトリをGitHubに公開しよう。GitHub上でNew repositoryを作り、pushまで完了させる。

💡 GitHubの画面に表示されるコマンドをそのままターミナルに貼ればOK！
TASK C — 発展

ブランチを使った機能追加

featureブランチを作ってCSSを追加し、mainにマージするまでの流れを実践しよう。

  1. git checkout -b feature/style でブランチ作成
  2. style.cssを作成してデザインを追加
  3. コミット後、mainに戻ってマージ
  4. GitHubにpushして確認
🏆 チャレンジ

Pull Requestを作成する

GitHubでフォーク→ブランチ作成→変更→プッシュ→プルリクエスト作成の流れをClaudeに聞きながら実践しよう。チームで開発する練習！
