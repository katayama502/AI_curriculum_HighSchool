演習課題 — Python

TASK A — 基礎

FizzBuzz

1から30までの数を表示するプログラム。3の倍数は「Fizz」、5の倍数は「Buzz」、両方の倍数は「FizzBuzz」と表示するプログラムをClaudeと一緒に作ろう。

💡 for + if/elif/else + % (余り)演算子を使う
TASK B — 応用

成績管理プログラム

科目名と点数を辞書に保存し、平均点・最高点・最低点を計算して表示するプログラムを作ろう。

  1. 科目:点数の辞書を作る
  2. values()で点数を取り出す
  3. sum() / len() で平均を計算
TASK C — 発展

じゃんけんゲーム

ユーザーが「グー/チョキ/パー」を入力するとコンピュータがランダムに選び、勝敗を判定するゲームをPythonで作ろう。

💡 input()でキーボード入力、random.choice()でランダム選択
🏆 チャレンジ

単語帳アプリ

英単語:日本語訳の辞書を作り、ランダムに英単語を出題→ユーザーが答えを入力→正誤判定を繰り返すシンプルな単語帳アプリを作ろう。
