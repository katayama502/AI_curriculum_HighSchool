リスト描画：map() と key

配列から要素を生成する

// 都市リストを表示するコンポーネント
const cities = [
  { id: 1, name: "東京", temp: 28 },
  { id: 2, name: "大阪", temp: 31 },
  { id: 3, name: "札幌", temp: 22 },
];

function CityList() {
  return (
    <ul>
      {cities.map(city => (
        <li key={city.id}>
          {city.name}: {city.temp}℃
        </li>
      ))}
    </ul>
  );
}
必須 key プロパティとは？
リスト要素には必ず一意の keyを指定すること。Reactが変更を効率よく検知するために使用します。
NG keyにindexを使うのは避けよう
key={index} は順番が変わると誤動作の原因に。データのID など変わらない値を使うこと。
TIP 条件付きレンダリング
{isLoading && <Spinner />} のように、論理演算子を使って表示を切り替えることができます。
