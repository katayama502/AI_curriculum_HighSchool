API活用 チートシート

🌐 HTTP基礎

# メソッド
GET    読み取り（変更なし）
POST   作成・データ送信
PUT    更新（全フィールド）
DELETE 削除

# ステータスコード
200 OK         成功
401 Unauthorized  APIキー未設定
404 Not Found  URLが存在しない
429 Too Many   レート制限超え

🐍 Python requests

import requests

# GET
res = requests.get(url, params={...})
data = res.json()

# POST with auth
res = requests.post(url,
  headers={"Authorization": f"Bearer {key}"},
  json={"key": "value"})

🤖 Claude API

import anthropic
client = anthropic.Anthropic()

msg = client.messages.create(
  model="claude-opus-4-6",
  max_tokens=1024,
  messages=[{
    "role": "user",
    "content": "質問テキスト"
  }]
)
print(msg.content[0].text)
