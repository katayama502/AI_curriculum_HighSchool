ハンズオン演習

1
Vercelにフロントをデプロイ
  • GitHubに最新コードをpush
  • Vercel → Add New Project → リポジトリ選択
  • VITE_API_URL を Railway URLに設定
  • Deploy → 生成URLを確認
2
Railwayにバックエンドをデプロイ
  • Railway → New Project → GitHub連携
  • Variables: JWT_SECRET / CORS_ORIGIN / NODE_ENV=production
  • Settings → Generate Domain をクリック
  • ヘルスチェック URL /api/health を確認
3
動作確認・デバッグ
  • ブラウザで公開 URL にアクセス
  • 新規登録・ログイン・CRUD を全て試す
  • Network タブでエラーが出たら確認
  • Vercel / Railway のログを確認
4
Lighthouse スコアを計測
  • Chrome → DevTools → Lighthouse タブ
  • Generate Report → Performance スコア確認
  • LCP / CLS の数値をメモ
  • 改善提案を 1 つ実施してみる
目標: 4つ全てクリアして、公開URLをクラスに共有しよう！
