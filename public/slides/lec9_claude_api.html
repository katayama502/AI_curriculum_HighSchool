Claude APIを使ってみよう

🔑 セットアップ手順
1
Anthropicのサイトでアカウント作成
console.anthropic.com にアクセス
2
APIキーを発行する
「API Keys」メニューから新規作成
3
anthropicライブラリをインストール
pip install anthropic
4
APIキーを環境変数に設定
ANTHROPIC_API_KEY=sk-ant-...
⚠️ APIキーの注意点
APIキーは絶対にGitHubに公開しないこと！
環境変数（.env）で管理しよう
💻 Claude APIの基本コード
import anthropic

# クライアントを初期化
client = anthropic.Anthropic()

# メッセージを送信
message = client.messages.create(
  model="claude-opus-4-5",
  max_tokens=1024,
  messages=[
    {
      "role": "user",
      "content": "こんにちは！"
    }
  ]
)

# 返答を表示
print(message.content[0].text)
