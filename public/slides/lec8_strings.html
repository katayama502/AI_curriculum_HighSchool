文字列の操作

f文字列（f-strings）が超便利

name = "花子" score = 95 # f文字列で変数を埋め込む print(f"{name}さんのスコアは{score}点です") # 計算も直接できる print(f"偏差値は約{score - 50}高いです") # 文字列のスライス（切り取り） text = "Hello World" print(text[0:5]) # "Hello" print(text[-5:]) # "World" print(len(text)) # 11

💡 f文字列の書き方：クォートの前に f を付け、変数を {変数名} で埋め込む。Python 3.6以降で使える最もモダンな書き方です。

よく使う文字列メソッド

  • .upper()全て大文字に変換
  • .lower()全て小文字に変換
  • .strip()前後の空白を除去
  • .replace(a, b)aをbに置換
  • .split(",")区切り文字で分割→リスト
  • .startswith(s)sで始まるか True/False
  • .count(s)sが何回含まれるか
  • .join(list)リストを文字列に結合
