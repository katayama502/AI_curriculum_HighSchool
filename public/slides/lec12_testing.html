テストをAIと書く

テストとは何か？

テストとは「コードが正しく動くことを自動で確認するプログラム」です。バグが入り込んだときに気づけます。
🔬
ユニットテスト
関数1つの動作を確認。例：「getTemp()は数値を返すか」
🔗
結合テスト
複数の部品を組み合わせた動作を確認
🖥️
手動テスト
ブラウザで実際に操作して確認する。高校生のうちはこれが中心

AIにテストを書いてもらう

テスト生成プロンプトと結果例
// プロンプト: calcTemperature()の
// テストをJestで書いて

import { calcTemperature } from './utils';

describe('calcTemperature', () => {
test('正常値: 25度を変換', () => {
expect(calcTemperature(25)).toBe(25);
});
test('null入力でエラー', () => {
expect(() => calcTemperature(null))
.toThrow('Invalid temperature');
});
});
