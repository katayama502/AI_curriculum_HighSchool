確認クイズ — Python基礎

Q1
Pythonで文字列に変数を埋め込むもっとも現代的な書き方は？
A. "名前は" + name
B. "名前は %s" % name
C. f"名前は{name}" ✓
D. "名前は".format(name)
f文字列（f-strings）はPython 3.6以降の推奨書き方。最もシンプルで読みやすい。
Q2
range(2, 10, 3) が生成する数列は？
A. 2, 3, 4, 5, 6, 7, 8, 9
B. 2, 5, 8 ✓
C. 3, 6, 9
D. 2, 4, 6, 8
range(start, stop, step)。2から始まり3ずつ増加、10未満まで → 2, 5, 8。
Q3
辞書から値を安全に取り出す（キーがなくてもエラーにならない）メソッドは？
A. dict.fetch(key)
B. dict.get(key) ✓
C. dict.find(key)
D. dict.search(key)
.get()はキーが存在しないときNoneを返す（エラーにならない）。デフォルト値も設定可能。
Q4
外部ライブラリをインストールするコマンドは？
A. python install numpy
B. pip install numpy ✓
C. import install numpy
D. get numpy
pipはPythonのパッケージ管理ツール。インストール後はimportして使う。
