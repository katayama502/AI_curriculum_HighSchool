辞書（dict）— キーと値のペア

名前をキーにデータを管理

# 辞書の作成 student = { "name": "田中太郎", "age": 17, "grade": "2年", "scores": [85, 92, 78] } # 値を取り出す print(student["name"]) # 田中太郎 print(student["age"]) # 17 # 値を更新・追加 student["age"] = 18 student["club"] = "バスケ部" # 辞書のループ for key, val in student.items(): print(f"{key}: {val}")

📋 APIのレスポンスデータ（JSON）は辞書と同じ構造。fetch APIで取得したデータも辞書として扱えます！

リスト vs 辞書

辞書：名前（"key"）でアクセス
