git tag — バージョンに名前をつける

タグとは？

タグは特定のコミットに「バージョン名」を貼り付ける機能です。v1.0, v2.0 のようにリリースポイントを明確にするために使います。

主なコマンド

git tag v1.0
現在のコミットにタグ「v1.0」をつける
git tag
タグの一覧を表示する
git push origin v1.0
タグをGitHubにpushする
git tag -d v1.0
タグを削除する
GitHubでは、タグが自動で「Releases」ページに表示され、ZIPダウンロードリンクも作られます。
