CSSでスタイルを付ける

/* セレクタ { プロパティ: 値; } */

/* タグセレクタ */
h1 {
  color: #339AF0;
  font-size: 32px;
}

/* クラスセレクタ */
.card {
  background: white;
  padding: 20px;
  margin: 10px;
}
📦 ボックスモデル
margin（外側の余白）
border（枠線）
padding（内側の余白）
content（コンテンツ）
よく使うプロパティ
color
文字色
background
背景色
font-size
文字サイズ
display:flex
横並びレイアウト
