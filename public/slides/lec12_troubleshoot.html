よくあるトラブルと解決パターン

困ったときの対処法

🔴 APIが動かない
TypeError: Cannot read property...
確認ポイント
APIキーが正しく設定されているか
URLのスペルミスがないか
ネットワーク接続はOKか
「APIが動かない。エラー: [エラー文]。URLとキーを確認して」
🔴 CSSが反映されない
スタイルが変わらない...
確認ポイント
ブラウザキャッシュをクリア（Ctrl+F5）
HTMLにstyle.cssが正しく読み込まれているか
クラス名・セレクタのタイポがないか
「style.cssのchangesが反映されない。HTMLの<link>タグを確認して」
🔴 git pushが失敗
error: failed to push some refs
確認ポイント
リモートに新しいコミットがあるか
まずgit pullして最新を取得
コンフリクトがあれば解消する
「git push でエラー。解決手順を教えて: [エラー文]」
🟡 AIの回答が的外れ
期待通りの修正にならない...
対処方法
/clearで会話をリセットする
より具体的なプロンプトに書き直す
ファイル名と行番号を明示する
「違います。〇〇ではなく△△をしてください。具体的には[詳細]」
🟡 コードが壊れた
修正後に動かなくなった...
対処方法
git checkout . で全変更を元に戻す
commitがあれば git reset HEAD~1
小さい変更から再実装する
「修正後に動かなくなった。修正前の状態に戻して」
🟡 npm installが失敗
EACCES: permission denied
確認ポイント
Node.jsが正しくインストールされているか
node -vでバージョン確認
Macなら sudo npm install -g
「npmエラー: [エラー文]。OSはMacです。解決して」
万能の対処法：困ったらまず「/clear」でリセット → エラー文を全文コピー → 「何をしたかったか」を一行で説明 → Claude Codeに貼り付ける。これだけで8割のトラブルは解決します！
