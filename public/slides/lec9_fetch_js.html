JavaScriptでAPI取得（fetch）

実践コード：天気データを取得して表示

// 無料API: Open-Meteo（APIキー不要） const URL = "https://api.open-meteo.com/v1/forecast" + "?latitude=35.68&longitude=139.69" + "¤t=temperature_2m,weathercode"; async function getWeather() { try { const res = await fetch(URL); if (!res.ok) { throw new Error(`HTTP Error: ${res.status}`); } const data = await res.json(); const temp = data.current.temperature_2m; document.getElementById("result") .textContent = `東京の気温: ${temp}℃`; } catch (err) { console.error("取得失敗:", err); } }

処理の流れ

1

URLを構築

ベースURL + クエリパラメータで場所や取得項目を指定

2

fetchでリクエスト

awaitで応答を待つ。非同期処理なのでasync関数の中で使う

3

res.ok でエラーチェック

ステータスコードが200以外のときはエラー処理をする

4

res.json()でデータ取得

JSON文字列をJSオブジェクトに変換して使う

5

DOMに表示

取得した値をtextContentで画面に反映させる

⚠️ try/catchは必須！ネットワークエラーや想定外の応答に備える
