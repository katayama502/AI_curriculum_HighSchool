ターミナルの基本操作

必須コマンド一覧

コマンド 意味 使い方の例
pwd 現在のフォルダを表示 → /Users/hayato/Desktop
ls ファイル一覧を表示 → フォルダの中身が見える
cd フォルダ名 フォルダに移動 cd myproject
cd .. 一つ上のフォルダへ → 親フォルダに戻る
mkdir 名前 新しいフォルダを作る mkdir my-app
touch ファイル名 新しいファイルを作る touch index.html
clear 画面をきれいにする → 出力がリセットされる
Tab補完：コマンド途中でTabキーを押すと自動補完！
↑キー：前のコマンドを呼び出せる。

実際の操作例

Terminal
~ $ cd Desktop
Desktop $ mkdir my-claude-app
Desktop $ cd my-claude-app
my-claude-app $ ls
# (空っぽ)
my-claude-app $ touch index.html
my-claude-app $ ls
index.html
my-claude-app $ claude
✓ Claude Code 起動！
Mac：アプリケーション → ユーティリティ → Terminal
Windows：スタート → 「PowerShell」と検索
