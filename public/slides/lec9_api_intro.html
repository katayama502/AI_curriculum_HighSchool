APIとは何か？

🍽️ レストランで例えると
お客さん
(クライアント/あなたのアプリ)
↕
ウェイター
(API = 窓口・仲介役)
↕
厨房
(サーバー/データ)
📌 ポイント
お客さんは厨房に直接入れない。
ウェイター（API）を通じて注文（リクエスト）し、料理（レスポンス）を受け取る。
APIの基本用語
エンドポイント
APIにアクセスするURL
リクエスト
APIへのデータ要求
レスポンス
APIからの返答データ
APIキー
認証用のパスワード
JSON
データのやり取りに使う形式
🌐 身近なAPIの例
・天気予報API → 今日の天気を取得
・YouTube API → 動画情報を取得
・Claude API → AIと会話する
