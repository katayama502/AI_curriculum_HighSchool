GitHub プロフィールを整えよう

  • 1
    プロフィール README を作る
    自分のユーザー名と同名リポジトリを作ると README がプロフィールに表示される（例: katayama/katayama）
  • 2
    ピン留めリポジトリを設定
    プロフィールに表示するリポジトリを最大6つピン留めできる。今回のアプリを入れよう
  • 3
    毎日 commit してグリーンを増やす
    Contribution Graph（緑のマス）が埋まっていると継続力が伝わる。小さい変更でもOK
  • 4
    アイコン・bio・場所を設定
    Settings → Profile から名前・プロフィール文・ウェブサイト URL を設定しよう
