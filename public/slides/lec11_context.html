Claude Codeはプロジェクト全体を理解する

Claude Codeが見ているもの

my-app/
👁 index.html ← 読んでる
👁 style.css ← 読んでる
👁 script.js ← 読んでる
👁 CLAUDE.md ← 最重要！
(node_modules/ は無視)
Claude Codeは起動するとプロジェクトの全ファイルを読み込み、「このプロジェクトが何をしているか」を自動で把握します。

CLAUDE.md — 説明書ファイル

CLAUDE.mdとは？
プロジェクトのルールや説明をClaudeに伝えるファイル。/initコマンドで自動生成できます。
# CLAUDE.md の例

## プロジェクト概要
高校生向け天気予報アプリ

## 技術スタック
- HTML / CSS / JavaScript
- Open-Meteo API

## コーディングルール
- コメントは日本語で書く
- 関数名は英語camelCase
ポイント
CLAUDE.mdが詳しいほど、Claude Codeの回答精度が上がる！チーム開発でも全員が同じルールで書けます。
