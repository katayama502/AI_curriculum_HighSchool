CSSのセレクタとプロパティ

「どこに」スタイルを当てるか — セレクタ

要素タグ名セレクタ
p { color: red; }
すべての <p> タグに適用される
クラスクラスセレクタ
.card { background: white; }
class="card" を持つ全要素に適用。最もよく使う
IDIDセレクタ
#header { font-size: 32px; }
id="header" を持つ1つの要素のみに適用

「何を」変えるか — 主要プロパティ

よく使うCSS プロパティ
color 文字色 color: #339AF0
background-color 背景色 background-color: white
font-size 文字サイズ font-size: 24px
padding 内側の余白 padding: 16px
margin 外側の余白 margin: 0 auto
border-radius 角を丸くする border-radius: 8px
display: flex 横並びレイアウト display: flex
