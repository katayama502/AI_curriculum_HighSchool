Tailwind CSSで素早くスタイリング

Tailwind CSSとは？

Tailwind CSSは「ユーティリティクラス」を組み合わせてスタイルを書くCSSフレームワーク。CSSファイルを別に書かずにHTMLに直接書ける。

❌ 通常のCSS（2ファイル必要）
/* style.css */
.btn { background: #339AF0; color: white;
padding: 8px 16px; border-radius: 6px; }
✅ Tailwind CSS（HTMLに直接）
<button className="bg-blue-500 text-white
px-4 py-2 rounded-md hover:bg-blue-600"
>追加する</button>
