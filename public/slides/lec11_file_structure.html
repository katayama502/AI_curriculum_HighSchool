Webプロジェクトのファイル構造

標準的なプロジェクト構成

my-weather-app/
├── index.html　　← メインページ
├── style.css　　　← デザイン
├── script.js　　　← 動作・ロジック
├── CLAUDE.md　　← AIへの説明書
├── .env　　　　　← APIキー（非公開）
├── .gitignore　　← Git除外設定
└── images/
　　├── icon-sun.png
　　└── icon-rain.png

各ファイルの役割

index.html
ページの骨格。「何を表示するか」の構造を定義する
style.css
見た目・デザイン。色・フォント・レイアウトを決める
script.js
動作・ロジック。APIを呼び出したりDOMを操作する
CLAUDE.md
Claude Codeへの説明書。プロジェクトのルールを書く
.env
APIキーを保存。.gitignoreに追加して絶対に公開しない
